# Contract Drafting Course Landing Page

A polished, modern landing page built with Vite and React for a contract drafting and freelancing course. The experience highlights the program’s value through a compelling hero section, trust indicators, curriculum highlights, pricing, and a strong call to action.

## Overview

This project presents a premium, conversion-focused landing experience for an online legal education program. It includes:

- A visually engaging hero section with animated content
- Trust and credibility indicators
- Course overview and benefits
- Why-join and unique-value sections
- Pricing information and a clear enrollment path
- A responsive footer and polished UI

## Tech Stack

- React 19
- Vite 8
- Framer Motion for animations
- Tailwind CSS for styling
- Lucide React for icons

## Project Structure

- src/components — reusable UI sections such as Hero, Pricing, Footer, and more
- src/App.jsx — main page composition
- public — static assets
- vercel.json — deployment configuration for Vercel

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment on Vercel

1. Push this repository to GitHub.
2. Open Vercel and import the project.
3. Vercel will detect the Vite app automatically.
4. Deploy using the existing build settings.

A Vercel configuration file is already included to support smooth single-page app deployment.

## Notes

The current implementation is designed to preserve the existing functionality while being production-ready for deployment.
