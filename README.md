# Lawctopus – Expert Course Landing Page Redesign

A modern, responsive, and conversion-focused landing page redesign for the **Lawctopus Law School – Expert-Level Course on "Mastering Contract Drafting and Freelancing."**

This project was developed as part of a frontend developer assessment, with the objective of redesigning the existing course page while preserving its core information architecture and significantly enhancing the user experience, visual hierarchy, and responsiveness.

## 📖 Project Overview

The original course page contains valuable information but follows a traditional long-form layout. This redesign focuses on creating a premium, modern user experience inspired by contemporary SaaS and EdTech websites while retaining the original course messaging and content flow.

The landing page has been designed with an emphasis on:

* Modern UI/UX principles
* Conversion-focused layout
* Mobile-first responsive design
* Smooth animations
* Clear content hierarchy
* Accessibility and performance
* Clean, reusable React components

---

## ✨ Features

* Responsive layout for desktop, tablet, and mobile
* Modern sticky navigation

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Navanish-Mehta/lawctopus-landing-page.git
```

### Navigate to the project

```bash
cd lawctopus-landing-page
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

---

## 🚀 Deployment

The project is deployed on **Vercel**.

To deploy your own copy:

1. Fork or clone the repository.
2. Push the project to GitHub.
3. Import the repository into Vercel.
4. Vercel will automatically detect the Vite framework.
5. Deploy using the default build configuration.
